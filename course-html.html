HTML
Hypertext Markup Language or HTML is the standard markup language used to create web pages and web applications. The history of HTML was created by Tim Berners-Lee, a physicist at the CERN research institute located in Switzerland. The first version of HTML was released by Tim Berners Lee in 1991, which had 18 tags. Since then, every time there is a new version, there will always be new tags and attributes (tag modifier). When working with this markup language, you use simple code structures (tags and attributes) to mark up web pages. Thanks to its ever-increasing popularity, this markup language is now considered an official web standard.

Get acquainted with tags in HTML

Tags are a prefix and suffix marker of an element in HTML. Tags are made with square brackets ( <...> ), then inside it contains the tag name and sometimes also added attributes. Example: < p > , < a > , < body > , < head > , and so on. Tags are always written in pairs.

Get to know Attributes

Attributes are additional arguments or accessories to an HTML element. Attributes are written only in the opening tag. And attributes function to add additional information to an HTML element
